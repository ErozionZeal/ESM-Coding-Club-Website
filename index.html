The ESM Coding Club

Purpose

This club is meant for ESM students who want to incorporate coding into their skillset. This club is supposed to be for beginners, however students familiar with code are completely welcome to join us.

About

This website is for the use of the members of the Esm coding club. It is meant to detail the Coursework and the materials that will be covered during the first semester of the ESM 2019-2020 school year. There will be links to the source code and general useful resources for code here.
